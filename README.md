WiseSpend is a simple expense tracker web application built using **React + Vite**.

## Features

- Light theme UI
- Dashboard with income, expense, balance
- Add expense form
- Expense history page
- Client-side routing using React Router

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Project Status

UI completed (Step 2).  
Logic & state management coming next.

## How to Run Locally

```bash
cd frontend
npm install
npm run dev
```
